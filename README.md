# DeepSeek-projection

## Quick start

You may need to set `PYTHONPATH` firstly:
```bash
$ cd deepseek-projection
$ export PYTHONPATH=${PYTHONPATH}:$(pwd)
```


1. Estimate the MFU of DeepSeek-V3/V2
```bash
$ python example/deepseek.py
```
